===============================================
          文档加密工具 v1.0 使用说明
===============================================

【功能介绍】
✅ 支持 PDF、Word(docx/doc)、Excel(xlsx/xls) 文档加密
✅ 采用国密 SM4 算法，安全可靠
✅ 自动密钥管理，加密记录可查询
✅ 支持批量加密文件夹中的所有文档

【使用方法】
1. 双击运行"文档加密工具v1.0.exe"
2. 点击"选择文件"选择要加密/解密的文件
3. 点击"加密文件"进行加密，加密后文件后缀为 .enc
4. 点击"解密文件"进行解密，程序会自动查找密钥
5. 点击"密钥记录"可查看所有加密记录

【批量加密】
点击"批量文件夹"可一次性加密整个文件夹中的所有文档

【注意事项】
⚠️ 密钥记录保存在程序同目录的 encryption_keys.json 文件中
⚠️ 请务必备份密钥记录文件，丢失后无法解密！
⚠️ 加密后原文件不会被删除，请自行处理
⚠️ 相同内容的文件会生成相同的密钥

【文件说明】
- 文档加密工具v1.0.exe：主程序
- encryption_keys.json：密钥记录（运行后自动生成）
- README.txt：本说明文件
